Q2: What is @ComponentScan used for?
@ComponentScan Spring mein isliye use hota hai: to automatically detect and register beans (classes annotated with @Component, @Service, @Repository, @Controller) from the specified package or its sub-packages into the Spring container. Example: @ComponentScan(basePackages = "com.rishi.project") Yeh bolega Spring ko: “Is package ke andar jitne bhi component classes hain, unko scan karke Spring container mein daal do.” Use it when your beans are in different packages and you want Spring to pick them up automatically.
